DEVELOPER'S NOTE

ABOUT THIS PROJECT

This project is made under personal interest. Web Framework used is Django. This WebApp has user specific data and made secure as much as possible, yet it is recommended to not to put serius data in the database. Database used is sqlite3 which is default functionality of Django. Application is deployed in Heroku. This is an open source project, source code is available in my GitHub. The website is not highly optimized for mobile devices.

TO DO LISTS

Click on Todolists to access your lists. You can create a new list but filling up name and description. Newly created list will show up immediately on the page. By clicking on it, you'll be redirected to the "items" page, where you can add a task. Any type of deletion is immediate and will affect the database permanently. In other words, Any deletion can not be UNDO; So be careful about what you delete.

LOGIN AND PAGE ACCESS

Todolists and items in it can only be accessed after authenticated login. If you are not registered, you first need to register. For now, if you forget your password, there's no password reset option, so you might need to remember it.
